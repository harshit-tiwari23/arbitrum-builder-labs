# ArbiLearn — Web3 & Arbitrum Explorer

A 4-page educational website built for Arbitrum Builder Labs (LamprosDAO).

## Pages
- **Home** — Landing page with Arbitrum/Layer 2 overview
- **Concepts** — Visual comparison cards (Web2 vs Web3, ETH vs BTC, etc.)
- **Live Prices** — Real-time crypto prices via CoinGecko API
- **Block Simulator** — Interactive proof-of-work mining simulator

## How to Run
Just open `index.html` in any browser. No install required.

## Known Issues / Future Improvements
- Live Prices may need a backend proxy for CORS in some environments
- Could add more coins and historical chart data
